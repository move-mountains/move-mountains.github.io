Hexo 更换主题

下载安装主题

以更换为 Next 主题为例。要先从 git 上将主题包下载下来，可以通过git clone的方式，也可以直接下载源码压缩包并解压。主题包的位置需要是${HEXO_HOME}\themes路径下。

下载解压(或者直接 clone)后，theme 就等于安装完毕，后面只需要启用。

启用主题

修改 Hexo 的配置文件_config.yml。找到theme这一块内容，改成想生效的主题（注意，要与 themes 里的文件夹名称相同）。
修改完成后保存，然后重启一下 Hexo，主题即可生效。

Hexo 正常启动后，命令行出现 ATTENTION

这个原因是 next 主题在 git 上的路径发生过迁移，老版本已不再维护，要获取最新版本的 NexT 主题需要前往新的 url（如 ATTENTION 中所示）。
这个坑的地方在于目前网上大多数教程里，下载 NexT 主题包时，使用的都还是老版本的 url。
老版本的包不是不能用，只是不再更新了。
